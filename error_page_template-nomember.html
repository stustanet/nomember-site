StuStaNet

Das StuStaNet ist der Verein, der in der Studentenstadt das Netzwerk betreut. Um im Internet surfen zu können, musst du den Proxy einstellen. Wenn du auch andere Online-Services wie z.B. Skype, ICQ, E-Mail oder Online-Gaming nutzen möchtest, musst du im StuStaNet Mitglied werden. Einen kostenlosen Internetzugang gibt es in der Bibliothek der StuSta.

Proxy einstellen

Solltest du bei der Einrichtung Probleme haben, so bitte einen Hausadministrator um Hilfe.

Ubuntu

Unter Ubuntu hast du die Möglichkeit, global einen Proxy zu definieren. So muss dieser nicht extra in jedem Browser eingetragen werden.

  1. Öffne die Netzwerk-Proxy-Einstellungen durch Klick auf System → Einstellungen → Netzwerk-Proxy.
  2. Hier markierst du ganz unten die Option Automatische Proxy-Konfiguration und trägst bei URL für Auto-Konfiguration: http://wpad.stusta.mhn.de/proxy.pac ein. Schließe nun das Fenster.

Mac OS

Unter Mac OS X hast du die Möglichkeit, global einen Proxy zu definieren. So muss dieser nicht extra in jedem Browser eingetragen werden, Firefox benötigt allerdings trotzdem eine gesonderte Einstellung

  1. Öffne die Netzwerkeinstellungen durch Klick auf Apfel (oben links) und wähle dann Systemeinstellungen → Netzwerk aus.
  2. Wähle links deinen Netzwerkadapter, klicke unten rechts auf den Button Weitere Optionen und wechsle auf die Registerkarte Proxies.
  3. Setze bei Zu konfigurierendes Protokoll vor Autom. Proxy-Konfiguration einen Haken und trage rechts bei URL http://wpad.stusta.mhn.de/proxy.pac ein. Schließe die Detaileinstellungen mit OK und bestätige erneut mit Anwenden

Browsereinstellungen

Unter Windows muss der Proxy in den Internetoptionen des Internet Explorers bzw. des alternativ verwendeten Browsers eingestellt werden.

Internet Explorer

  1. Internet Explorer starten.
  2. Wähle im Untermenü Extras den Punkt Internetoptionen.
  3. Im Reiter Verbindungen auf den Button LAN-Einstellungen klicken.
  4. Setze bei Automatisches Konfigurations-Skript verwenden einen Haken und trage bei der Adresse http://wpad.stusta.mhn.de/proxy.pac ein.
  5. Bestätigen Sie mit OK und schließen Sie die restlichen Fenster.

Mozilla Firefox

  1. Firefox starten.
  2. Wähle im Untermenü Extras den Punkt Einstellungen. Unter Linux befinden sich die Einstellungen unter Bearbeiten → Einstellungen.
  3. Klicke unter der Kategorie Erweitert auf den Reiter Netzwerk.
  4. Im Bereich Verbindung auf den Button Einstellungen… klicken.
  5. Markiere den untersten Punkt und trage als Automatische Proxy-Konfigurations-URL: http://wpad.stusta.mhn.de/proxy.pac ein.
  6. Bestätige mit OK und schließe die restlichen Fenster.

Virus infection

Your internet connection has been blocked because your computer is infected with viruses or other malware. Viruses pose a huge threat to the Munich Scientific Network, which the network of the Studentenstadt is a part of. Infected computers cause large scale damage a and have significant impact on the function of the network. Some viruses steal credit card data from online shopping. Others are use your computer to automatically attack other networks or computers.

Further procedure

If you can plausibly explain how you got rid of all of the malware, an release of the blocking is possible. In case of another incident you will be subject to a permanent ban by the Studentenwerk.

You need to complete the following tasks in order to get your network connection reactivated:

  • immediately disconnect your computer from the network to prevent further damage
  • virus scan with a rescue CD (links to virus scanners)
  • remove all viruses (often impossible; in that case a new installation of the computer is unavoidable)
  • install all availiable updates for your operating system
  • install a virus scanner
  • keep all software, most importantly operating system, virus scanner and web browser, up to date

Unfortunately virus scanners today are incapable of reliably recognising all viruses. It is therefore possible, that a virus cannot be found or removed.
In that case or if you do not have the time or know how to maintain your Windows System secure and free of malware, please seriously consider switching to a better operating system like Ubuntu (free download). This is very easy to use, free and offers more features than Windows and is much more secure.

Help

In the case you are in need of some help, please ask an administrator.

Application for reactivation

You can then ask with an email to novirus@stusta.mhn.de for reactivation. You need to include all of the following in order get it handled:

  • Name
  • Number of building and apartment you live in
  • IP address
  • extensive description of what you have done to insure that your compoter poses no longer any threat, including: the names of the removed viruses and the virus scanner in use or the complete log of the virus scanner.

There is free internet access in the StuSta library.

Please note:
Only send an email if you are absolutely sure the problem has been solved. In case of more than one email all of them will be ignored. The office of the StuStaNet wont be able to tell you more than what is written here.
Emails "I've solved the problem" or "I deleted the virus" will be ignored, too.
